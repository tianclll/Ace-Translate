# AceTranslatePro UI Visual Refinement Plan

## 问题诊断
- **窗口过大**：1000×720 → 900×680（已确认）
- **图标丑陋**：emoji 作图标显得廉价（等用户下载 SVG）
- **视觉噪音多**：虚线边框、太多 emoji 标签、颜色层次太平均
- **间距浪费**：内边距 20-24px 过大，内容区利用率低

## 设计方向

**一句话定位**：精准的翻译工具，界面应该像它的翻译结果一样干净。

### 色彩系统（优化现有青绿主题）

| Token | 当前值 | 新值 | 说明 |
|---|---|---|---|
| `--bg` | `#F0FDFA` | `#F8FAFA` | 更极简的背景，减少色偏 |
| `--card-bg` | `#FFFFFF` | `#FFFFFF` | 卡片白底保留 |
| `--primary` | `#0D9488` | `#0B7C72` | 稍微加深，提升可读性 |
| `--border` | `#CCFBF1`→`#E5E7EB` | `#E8ECEF` | 统一为中性灰，降低视觉重量 |
| `--text` | `#134E4A` | `#1A1A2E` | 更深的阅读色 |
| `--muted` | `#6B7C78` | `#889096` | 次要文字更淡 |
| `--accent` | `#EA580C` | 移除 | 橙色强调不适合工具产品 |

### 排版
- 标题：16px bold → 15px semibold（降低标题压迫感）
- 正文输入：15px → 14px（更紧凑）
- 导航图标：22px → 18px（缩小）
- 标签文字：10px → 11px（提高可读性）

### 布局调整
- 导航栏宽度：72px → 64px（省空间）
- 内容区内边距：24px → 20px
- 卡片内边距：20px → 16px
- 翻译双栏间距：20px → 16px

### 签名元素
左上角 logo 区域 — 圆形深色底 + 白色 "A" 字，作为视觉锚点。

### 精简装饰
1. 移除虚线边框 → 改用 1px 实线细边
2. 移除所有 emoji + 文字混合的按钮标签 → 纯文字按钮（等 SVG 图标到位再换）
3. 原文/译文面板的区分：左栏白底细灰边，右栏浅青底 + 左边框强调线（而非全边框）
4. 语言切换条简化：去掉多余 emoji，用更紧凑的布局
